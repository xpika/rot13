rot13
=====

encryption cipher


usage:

from haskell

```haskell

Prelude Codec.Encryption.Rot13> encrypt "The Quick Brown Fox Jumps Over The Lazy Dog"
"Gur Dhvpx Oebja Sbk Whzcf Bire Gur Ynml Qbt"
```

from shell

```bash
$rot13
The Quick Brown Fox Jumps Over The Lazy Dog
Gur Dhvpx Oebja Sbk Whzcf Bire Gur Ynml Qbt
```
